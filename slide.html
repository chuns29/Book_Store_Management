1. Tổng Quan Đề Tài

Đặt Vấn Đề

Nhu cầu mua sắm trực tuyến ngày càng tăng cao, đặc biệt là các sản phẩm văn hóa như sách. Các cửa hàng truyền thống gặp khó khăn trong việc quản lý kho, tiếp cận khách hàng và thống kê doanh thu thủ công.


Mục Tiêu

Xây dựng hệ thống BookStore hoàn chỉnh giúp:

  • Người dùng dễ dàng tìm kiếm và mua sách online.
  • Quản trị viên quản lý kho hàng, đơn hàng và báo cáo doanh thu hiệu quả.

2. Công Nghệ Sử Dụng

Hệ thống được xây dựng trên nền tảng MERN Stack hiện đại:

ReactJS

Frontend tương tác mượt mà (SPA).

Node.js

Xử lý logic Server mạnh mẽ.

Express

Framework xây dựng RESTful API.

MongoDB

Cơ sở dữ liệu NoSQL linh hoạt.

Tailwind CSS JWT Auth Multer XLSX Import

3. Kiến Trúc Hệ Thống

Mô hình Client-Server

  • Frontend: Giao tiếp với Backend thông qua các API (Fetch).
  • Backend: Nhận request, xử lý logic, tương tác Database và trả về JSON.
  • Database: Lưu trữ Users, Products, Orders.
Điểm nổi bật:

Sử dụng cơ chế xác thực JWT (JSON Web Token) giúp bảo mật phiên đăng nhập mà không cần lưu session phía server (Stateless).

Dương Quang Chung - 20225266 4/10

4. Chức Năng Người Dùng (Khách)

Tìm Kiếm & Lọc

Tìm sách theo tên, tác giả. Bộ lọc thông minh theo thể loại (Fiction, Technology...), sắp xếp theo giá, bán chạy, đánh giá cao.

Giỏ Hàng & Đặt Hàng

Thêm sản phẩm, điều chỉnh số lượng, xóa sản phẩm. Quy trình đặt hàng (Checkout) kiểm tra tồn kho theo thời gian thực.

Đánh Giá & Bình Luận

Chỉ cho phép đánh giá khi đã mua hàng. Tính điểm sao trung bình tự động.

Tài Khoản

Đăng ký, đăng nhập, xem lại lịch sử đơn hàng và trạng thái đơn hàng.

Dương Quang Chung - 20225266 5/10

5. Chức Năng Quản Trị (Admin)

  • Quản Lý Sản Phẩm: Thêm, Sửa, Xóa sách. Cập nhật thông tin chi tiết (Giá, Tồn kho, Tác giả...).
  • Import Excel: Tính năng cao cấp cho phép nhập hàng loạt dữ liệu sách từ file Excel, tiết kiệm thời gian nhập liệu.
  • Dashboard Thống Kê:
    • Tổng doanh thu.
    • Số lượng đơn hàng.
    • Top khách hàng thân thiết.
